# **P02: Clasificación de Cáncer de Pulmón mediante Modelos de Machine Learning y Optimización Bayesiana**
Este proyecto tiene como objetivo desarrollar y comparar distintos modelos de clasificación para predecir la presencia de cáncer de pulmón a partir de variables relacionadas con hábitos y síntomas. 
Se implementaron tres arquitecturas de aprendizaje automático: Regresión Logística, SVM con kernel RBF y Multi-layer Perceptron (MLP), optimizando sus hiperparámetros mediante Optimización Bayesiana.
El proceso incluyó preprocesamiento de datos, ingeniería de características, escalamiento y validación cruzada (K-Fold), utilizando el F1-score como métrica principal de evaluación.
Los resultados obtenidos muestran un alto desempeño de los modelos, con valores de F1 cercanos a 0.95, destacando la importancia de una adecuada preparación de los datos.
